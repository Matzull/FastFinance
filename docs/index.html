Key Features

Multi-Account

Manage multiple bank accounts, track balances across currencies and institutions.

Transactions

Log income and expenses with categories, notes, and bulk import from bank statements.

Subscriptions

Monitor recurring payments and calculate monthly/annual costs automatically.

Net Worth

Track assets, liabilities, and see your total net worth evolve over time.

Insights

Financial analysis, spending patterns, and personalized recommendations.

Telegram Bot

Chat interface with OCR support to send receipts and log transactions on-the-go.

Mobile App

Native Kivy/KivyMD app for Android and desktop—full feature parity.

REST API

Complete REST API with OpenAPI documentation and SDK support.

Self-Hosted

Deploy on your own server or local machine. Full data privacy and control.

Architecture

Web UI

FastAPI + Jinja2 templates. Responsive dashboard with charts, transaction viewing, and account management.

Run: uv run fastfinance-web

Telegram Bot

Chat interface with PaddleOCR / OpenAI Vision integration. Send receipt photos and log transactions instantly.

Run: uv run fastfinance-bot

Mobile App

Kivy + KivyMD native client for Android and desktop. Full feature parity with web UI.

Run: uv run fastfinance-mobile

Database

SQLAlchemy with SQLite by default. Models for banks, transactions, subscriptions, budgets, and more.

Location: patrimonio.db

CLI

Typer + Rich CLI for managing banks, transactions, subscriptions from the terminal.

Run: uv run fastfinance --help

REST API

Full REST API at /api/. Interactive docs at /docs.

Use: FastAPI/OpenAPI clients or webhooks

Quick Setup

1

Clone & Install

git clone https://github.com/YOU/patrimonio.git
cd patrimonio
uv sync
2

Run Web App

uv run fastfinance-web
# Open: http://127.0.0.1:8000
3

(Optional) Telegram Bot

export TELEGRAM_BOT_TOKEN="your-token-here"
uv run fastfinance-bot
4

(Optional) Mobile App

uv pip install -e ".[mobile]"
uv run fastfinance-mobile

Node: FastFinance requires Python 3.12+. Use uv for dependency management (it handles virtual environments automatically).
